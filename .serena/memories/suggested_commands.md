# Suggested commands
- List files: `find . -maxdepth 3 -type f -print`
- Check git status: `git status --short`
- If package tooling is added later: inspect `package.json` scripts and run the relevant `npm run lint`, `npm run test`, or `npm run build`.
- Chrome extension manual load flow: open `chrome://extensions`, enable Developer mode, click Load unpacked, select the project directory.
